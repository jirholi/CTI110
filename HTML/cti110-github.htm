What is GitHub?

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. This tutorial teaches you GitHub essentials like repositories, branches, commits, and Pull Requests.

Getting Started with GitHub

  1. Create a GitHub account at github.com.
  2. Learn the GitHub flow: Basic GitHub workflow.
  3. Try creating your first repository and practice making commits.

HTML Basics

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

Simple HTML Page Structure

                
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Try creating your own HTML page with this structure and modify the contents!
